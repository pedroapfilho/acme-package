A minimal external store with a typed `get` / `set` / `subscribe` surface.

```ts
import { createStore } from "@acme/core";

const store = createStore({ count: 0 });

store.subscribe(() => console.log("changed:", store.get()));
store.set((previous) => ({ count: previous.count + 1 }));
```

## API

- `createStore<T>(initial)` — returns a `Store<T>`.
- `get()` — returns the current state.
- `set(next | updater)` — replaces the state with a value, or with the result
  of an updater receiving the previous state. Listeners are skipped when the
  next state is `Object.is`-equal to the current one.
- `subscribe(listener)` — registers a listener called after every state
  change; returns an unsubscribe function.

The zustand-style caveat applies: because `set` treats a function argument as
an updater, function-valued state must be wrapped — `set(() => fn)`, not
`set(fn)`.
